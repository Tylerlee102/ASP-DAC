# Rebuttal Preparation

The project should rebut from evidence, not rhetoric.

## Core Position

ReplayCapsule-RV is not a generic trace compressor, runtime monitor, or replay system. The intended contribution is the defined and evaluated event boundary for deterministic replay of embedded RV32I failures under interrupts and MMIO.

## Evidence To Accumulate

- novelty matrix with unsafe claims removed
- formal event model and sufficiency theorem
- RTL capture path and property checker
- firmware-running bug suite
- replay success table
- baseline trace-size table
- ablation table
- synthesis/resource table
- limitations table

## Claims To Avoid

- first ever deterministic replay
- first ever hardware monitor
- general minimality for all systems
- multicore/cache/DMA support
- ASIC tapeout or FPGA measurements not actually performed
